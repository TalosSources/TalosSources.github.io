Préparation

  • Versez les céréales dans un bol de taille appropriée.
  • Versez le lait par dessus les céréales. Faites selon vos préférences, mais une méthode raisonnable est de s'arrêter de verser dès que la surface du lait devient visible.
  • optionel: vous pouvez si vous le souhaitez attendre une minute ou deux afin de laisser vos céréales rammolir avec le lait.
